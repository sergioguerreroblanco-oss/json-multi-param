Detailed Description

Abstract base class for all parameters.

ParamBase defines the type-erased interface used by ParamSet. Concrete implementations (Param<T>) provide typed storage, constraints checking, and parsing/formatting logic.

Copy and move operations are disabled to keep a clear single-owner model and avoid accidental slicing or ownership confusion when used through base pointers.
