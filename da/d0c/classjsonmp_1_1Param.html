Detailed Description

template<typename T>
class jsonmp::Param< T >

Strongly-typed parameter with constraints.

Template Parameters
TParameter value type.

Param<T> stores a value of type T and enforces validation rules through Constraints<T>. It provides string-based conversion for generic serialization and parsing.
