ParamBase provides the non-templated interface used by ParamSet to store and handle heterogeneous typed parameters (Param<T>) in a single container.

A parameter has:

  • A stable logical name (key).
  • A logical value type (ValueType).
  • String-based conversion for serialization and parsing.
